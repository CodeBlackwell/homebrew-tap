# Codeblackwell Tap

## How do I install these formulae?

`brew install codeblackwell/tap/<formula>`

Or `brew tap codeblackwell/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "codeblackwell/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
